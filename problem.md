# Matrix Sum 22725
You are given a square matrix A of size n × n.
All elements of the matrix are equal to x.
Your task is to compute A^p and return the sum of all elements of A^p modulo 10^9 + 7.
## Input
t — number of test cases
Each test case contains:
n x p
## Constraints
1 ≤ t ≤ 10^4
1 ≤ n ≤ 10^9
0 ≤ x ≤ 10^9
1 ≤ p ≤ 10^9
## Example
Input
1
2 2 3
Output
128
